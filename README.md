# ControllerToMouse
https://www.youtube.com/watch?v=7eX8Zdy9HLI

Features:
Move mouse around on screen and supports multiple screens
Dragging things around
Sensivity settings
Triangle is right mouse click
X is left mouse click
L3 is middle mouse click
R3 refresh page
L1 and R1 scroll up and down
Hold L2 to activate move divider
Hold R2 to activate move multiplier
Share button to toggle on-screen keyboard
Psn button is windows key
Square is media forward
Circle is media backwards
Touchpad is pause/play
Options button is escape
